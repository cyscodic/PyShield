# PyShield – AI Powered Secure Code Analysis Platform

PyShield is a robust, production-ready SaaS platform that performs AST-based, AI-powered secure code analysis on Python codebases. It integrates seamlessly with GitHub, provides comprehensive PDF reports, and features a modern dashboard for vulnerability review and remediation.

## Architecture
- **Frontend**: Next.js (App Router)
- **Backend**: FastAPI (Python 3.10+)
- **Database**: PostgreSQL (SQLAlchemy)
- **AI Integration**: Google Gemini API
- **Infrastructure**: Docker & Docker Compose

## Repository Structure
This is a monorepo containing both the frontend and backend services.

- `/frontend` - Next.js web application
- `/backend` - FastAPI server and AST scanner engine
- `/docs` - Project documentation and architecture diagrams
